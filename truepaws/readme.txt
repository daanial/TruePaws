Self-hosted Kennel Management System for professional breeders. Manage animals, lineage, reproduction, and sales with automated PDF generation.

== Description ==

TruePaws is a comprehensive, self-hosted kennel management plugin for WordPress, designed specifically for professional dog and cat breeders. It provides a modern, app-like interface for managing animal husbandry, health records, reproduction tracking, and sales.

= Features =

* Animal management with photos, lineage, and health history
* Automatic pedigree generation and PDF certificates
* Litter management with mating logs and pregnancy calculator
* Whelping wizard for batch puppy creation
* Contact management for buyers and waitlists
* Shortcodes for website integration

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/truepaws/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Visit TruePaws in your WordPress admin menu to get started
